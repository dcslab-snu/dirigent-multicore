# dirigent-multicore

How to build and run it?
---
$ cd ${DIRIGENT_HOME}/cmr/pcm  
$ make  
$ cd ${DIRIGENT_HOME}/cmr  
$ make  

To run dirigent with freq. scale and dynamic partition enabled.  
$ sudo ./cmr ${cfg} freq part  

or To run dirigent to get offline solorun data  
$ sudo ./cmr ${cfg} no no
